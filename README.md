# 💀 mansonware-portfolio

Portfólio pessoal com tema **hacker/cybersecurity** — uma página única em HTML/CSS/JavaScript puro, com fundo estilo *Matrix* e uma sequência de *boot* animada.

> Cybersecurity • Linux • Automation • Ethical Hacking

---

## ✨ Funcionalidades

- 🟢 **Fundo Matrix animado** desenhado em `<canvas>`, responsivo ao tamanho da janela.
- ⌨️ **Boot sequence** com efeito de digitação linha a linha e cursor piscante.
- 📱 **Layout responsivo** com tipografia fluida (`clamp()`).
- ♿ **Acessibilidade**: região `aria-live` para o log de boot e respeito a `prefers-reduced-motion`.
- 🔗 Botão de acesso ao perfil do GitHub.

## 🛠️ Tecnologias

- HTML5
- CSS3 (variáveis, `clamp()`, media queries)
- JavaScript (sem dependências externas)

## 🚀 Como executar

Não há build nem dependências. Basta abrir o arquivo:

```bash
git clone https://github.com/Mansonware/mansonware-portfolio.git
cd mansonware-portfolio
```

Depois é só abrir o `index.html` no navegador, ou servir localmente:

```bash
# Python 3
python3 -m http.server 8000
# acesse http://localhost:8000
```
